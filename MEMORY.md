# Agent Memory

This file is a rolling memory for agents working on this project.
Use it to capture concise lessons from implementation feedback so similar future tasks can be handled better.

## Reusable Rules

- Center this project on the sequencer module demonstration as the primary deliverable.
- Keep orchestration strictly message-driven through Protocol; never use direct module-to-module calls.
- Prioritize timing precision and reliability in sequencer design decisions.
- Support runtime sequence ingestion and both single-run and continuous execution modes.
- Include wait, decision/branch, and returned-value reporting capabilities in core sequencing behavior.
